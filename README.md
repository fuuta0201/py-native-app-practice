# py-native-app-practice

## venv

### create venv

```
python3.12 -m venv myenv
```

### activate

```
source myenv/bin/activate
python -V
```

### deactivate

```
deactivate
```

## libraries

### install

```
pip install -r requirements.txt
```

### update requirements.txt

```
pip freeze > requirements.txt
```
